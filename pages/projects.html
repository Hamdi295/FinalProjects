Engineer Final Project

As part of our final year project, we implemented a DevOps solution within KMF. To achieve this, we created an infrastructure using VMware ESXi with six virtual machines: one for GitLab, one for Jenkins, one for the staging environment, one for pfSense, one for Redmine, and one for SonarQube. We configured these VMs to enable continuous integration with webhooks and a container registry, ensuring smooth coordination across different stages of development. Additionally, we developed scripts to streamline the overall process and reduce human errors.Furthermore, we integrated Kubernetes into the staging VM, enabling efficient management of application deployment and scaling. This setup enhances our DevOps cycle by facilitating collaboration, automation, and code quality, while ensuring optimal security and project management.
